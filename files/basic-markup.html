A (very) basic HTML template

<!DOCTYPE html>
<html lang="en">
   <head>
      <meta charset="utf-8">
      <title>My first webpage</title>
      <link rel="stylesheet" href="#">
   </head>
   <body>
      <h1>My first webpage</h1>
      <p>Hello, this is my first webpage.</p>
   </body>
</html>

Copy and paste the code above into your code editor, and save it as a new file called index.html. Edit the content as you wish and then upload the file to the public_html folder using FileZilla.
